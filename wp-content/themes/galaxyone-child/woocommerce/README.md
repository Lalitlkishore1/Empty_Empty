# WooCommerce Template Override Policy

GalaxyOne Child does not override WooCommerce templates in Phase 3.

WooCommerce presentation is handled through `assets/css/woocommerce.css`. Product behavior, pricing, inventory, cart, checkout, orders, delivery, offers, and rewarded advertisements remain outside the child theme.

A WooCommerce template override may be added only when a later approved phase requires a presentation change that cannot be achieved through supported WooCommerce hooks, Elementor layout controls, or CSS. Any override must be documented and reviewed for WooCommerce upgrade compatibility.
